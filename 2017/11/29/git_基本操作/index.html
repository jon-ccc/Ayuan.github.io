git_基本操作

发表于 2017-11-29 | 分类于 git

文档整体解决方案(readthedocs、github 、sphinx)使用

chrome github 插件:Octotree

github初始设置

添加&新建一个github仓库到本地
新建一个README.md

echo "# Ayuan" >> README.md

初始化

git init
git add README.md
git commit -m "first commit"

添加远程项目到本地

git remote add origin git@github.com:jon-ccc/Ayuan.git

上传本地的代码到服务器

git push -u origin master

把服务器项目下载到本地

git pull git@github.com:jon-ccc/jon.git

克隆一个项目

git clone git@github.com:jon-ccc/jon.git

三步上传

git add . #添加所有改动过的文件
git commit -m . #提交所有改动过的文件
git push

关联github并上传本地的仓库

git remote add origin git@github.com:jon-ccc/Ayuan.git
git push -u origin master
